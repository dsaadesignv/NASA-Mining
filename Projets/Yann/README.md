Near Earth Object
Yann SAINT-CLAIR
Description

NEO est une interface qui va mettre de récupérer les données de l'API NEO, de les visualiser et de les comparer avec d'autre valeurs ayant plus de sens à une échelle terrestre.
Le ton ce veut plutot coloré est amusant. Néanmoins, il ne m'a pas été possible d'aller jusqu'à la finalisation complète de mon projet, il faudrait que je réalise d'avantage
de point de comparaisons et que j'y ajoute des animations de l'interface ainsi qu'un dimension sonore avec un travail de sound design.
